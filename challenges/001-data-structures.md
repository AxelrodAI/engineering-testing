# Challenge 001: Core Data Structures Library

**Difficulty:** Beginner
**Goal:** Build a tested library of fundamental data structures from scratch.

## Requirements

Implement in JavaScript (Node.js, ESM):

1. **Stack** — push, pop, peek, isEmpty, size, toArray
2. **Queue** — enqueue, dequeue, front, isEmpty, size, toArray
3. **LinkedList** — append, prepend, delete, find, toArray, size, reverse
4. **HashMap** — set, get, delete, has, keys, values, size (handle collisions)

## Rules
- No external dependencies (only Node built-ins)
- 100% test coverage for all public methods
- Handle edge cases (empty structures, duplicates, large inputs)
- Performance: document time complexity for each method

## Success Criteria
- All tests pass
- Edge cases covered
- Clean, readable code
- JSDoc comments on public methods
